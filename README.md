## 프로젝트 소개

- 고도 청각장애인을 위한 소음 측정 및 위험도 분석과 대화 감정 분석을 주 기능으로 한 웹 앱
- 소음 측정 및 위험도 분석 - 소음을 측정하여 어떠한 소음인지 나타내어 주고 위험도에 따라 푸시 알림
- 대화 감정 분석 - 음성을 실시간으로 STT 하는 동시에 어조를 통한 감정을 예측하여 출력
